# Research Claims: Knowledge Management

Backed by cognitive science and tool-for-thought research.

## Core Claims

### 1. Spreading Activation
**Claim:** Wiki-links enable associative discovery through spreading activation.

**Research:** Network theory in cognitive science
**Implication:** Dense linking improves knowledge retrieval

### 2. Generation Effect
**Claim:** Writing reinforces memory better than re-reading.

**Research:** Slamecka & Graf (1978)
**Implication:** Extract insights rather than copy

### 3. Context Switching Cost
**Claim:** Switching contexts incurs cognitive overhead.

**Research:** Leroy (2009)
**Implication:** MOCs reduce navigation overhead

### 4. Atomic Notes
**Claim:** One idea per note improves composability.

**Research:** Zettelkasten methodology
**Implication:** Enables flexible recombination

### 5. Progressive Disclosure
**Claim:** Description fields aid navigation without overwhelming.

**Research:** Nielsen Norman Group
**Implication:** Always include summaries

### 6. Fresh Context Per Task
**Claim:** LLM attention degrades as context fills.

**Research:** Anthropic documentation
**Implication:** Use subagents for multi-phase tasks

### 7. Session Boundaries
**Claim:** Clear session boundaries improve learning.

**Research:** Agent architecture patterns
**Implication:** Capture state at session end
